## Coding manual (v1)

This folder is reserved for the **coding manual** required before the pilot.

- `coding_manual_v1.md`: Coding Manual v1.0 (annotator training + reference)
- `coding_manual_v1.doc`: Word-openable text copy (lightweight)
- `coding_manual_v1_outline.md`: minimum structure + authoring checklist (historical/scratchpad)
- `examples/`: worked JSON examples used for training and calibration (incl. gold standards placeholder)
