# B02: تباين تفسير الربا

> **السؤال**: قارن تفسيرات المصادر الخمسة للربا عبر كل آية يظهر فيها. أنشئ مصفوفة تباين توضح أين يتفقون ويختلفون.

---

## القدرات المطلوبة

- `TAFSIR_MULTI_SOURCE`: مصادر تفسير متعددة
- `CONSENSUS`: تحليل الإجماع
- `PROVENANCE`: إثبات المصدر

---

## المصادر المقارنة

1. **ابن كثير** - تفسير بالمأثور
2. **الطبري** - تفسير تاريخي شامل
3. **القرطبي** - تفسير فقهي
4. **السعدي** - تفسير روحي معاصر
5. **الجلالين** - تفسير مختصر

---

## الآيات الرئيسية

| الآية | الموضوع |
|-------|---------|
| 2:275 | تحريم الربا الأساسي |
| 2:276 | محق الربا |
| 2:278-279 | التحذير الشديد |
| 3:130 | أضعاف مضاعفة |
| 30:39 | الربا لا يربو عند الله |

---

## مصفوفة التباين

| الموضوع | ابن كثير | الطبري | القرطبي | السعدي | الجلالين |
|---------|----------|--------|---------|--------|----------|
| تعريف الربا | ✓ | ✓ | ✓✓ | ✓ | ✓ |
| أنواع الربا | ✓ | ✓✓ | ✓✓ | ✓ | - |
| الحكمة | ✓ | ✓ | ✓ | ✓✓ | - |
| العقوبة | ✓✓ | ✓ | ✓✓ | ✓ | ✓ |

**ملاحظة**: ✓✓ = تفصيل موسع، ✓ = ذكر، - = لم يذكر

---

## الخلاصة

التباين الرئيسي في:
- **القرطبي**: أكثر تفصيلاً في الأحكام الفقهية
- **الطبري**: أكثر تفصيلاً في الروايات
- **السعدي**: أكثر تركيزاً على الحكمة الروحية
